# LearnFlow

Personalized Learning Roadmap - AI-powered platform for skill gap analysis, learning roadmaps, and interview preparation.

## Project Structure

- **frontend/** - React + TypeScript frontend with TanStack Router
- **backend/** - Express + TypeScript REST API with Prisma ORM

## Getting Started

### Frontend
```bash
cd frontend
pnpm install
pnpm dev
```

### Backend
```bash
cd backend
pnpm install
pnpm prisma migrate dev
pnpm dev
```

## Features

- AI-powered resume analysis with company-level strictness
- GitHub profile integration and repository analysis
- Skill gap identification and personalized learning roadmaps
- Career coach recommendations
- Assessment and interview preparation
